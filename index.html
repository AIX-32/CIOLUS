02 - EXECUTION
Multi-Step Agent

It keeps going
until the
job is done.

One message can trigger dozens of tool calls. CLIO works iteratively - inspecting, deciding, acting - up to 40 steps per task. A stop button is always present.

When you say "reorganise my Downloads folder by type", CLIO first lists the directory, then creates subdirectories, then moves each file. No prompting required between steps.

All tool calls are logged in the sidebar. You see every action as it happens. The operation can be interrupted at any point.
